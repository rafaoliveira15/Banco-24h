## Sobre o Projeto

O Bank 24h é uma aplicação de simulação bancária desenvolvida para o curso de Técnico em Desenvolvimento de Sistemas no SENAI "A. Jacob Lafer". O sistema foi projetado para operar como um terminal multibancos, permitindo a gestão de contas de seis instituições diferentes (Itaú, Santander, Nubank, Bradesco, Caixa e Banco do Brasil) em uma única interface dinâmica.

O foco do projeto foi a aplicação de lógica de programação avançada em JavaScript, manipulação de estados do DOM e segurança de dados do usuário.

## Funcionalidades

* **Interface Adaptativa:** Alteração automática de cores, logotipos e estilos de acordo com o banco selecionado.
* **Sistema de Autenticação:** Login validado com base em objetos de dados seguros.
* **Operações de Caixa Eletrônico:**
    * Saques e depósitos com atualização imediata de saldo.
    * Transferências bancárias entre contas.
    * Consulta de extrato detalhado das operações.
* **Segurança de Dados:** Módulo de alteração de senha com validações via REGEX e estruturas de repetição para garantir integridade.
* **Saída de Documentos:** Geração de comprovantes e extratos em formato PDF via biblioteca jsPDF.

---

## Planejamento e Metodologia

O desenvolvimento seguiu o Product Backlog dividido em quatro etapas principais:

1. **Design e Estrutura:** Criação da identidade visual e estrutura HTML/CSS.
2. **Lógica de Negócio:** Implementação das funções financeiras e validações de saldo.
3. **Integração de PDF:** Desenvolvimento do módulo de exportação de comprovantes.
4. **Documentação:** Consolidação dos artefatos técnicos e revisão de código.
